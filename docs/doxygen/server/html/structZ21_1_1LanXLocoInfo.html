uint8_t Z21::LanXLocoInfo::functionIndexMax ( ) const
inline

Get maximum fuction index stored in this message.

Returns
Maximum index
Note
There is also a function at index 0 so count it

Maximum function index depends on Z21 firmware protocol version versions < 1.42 support up to F28 versions >= 1.42 support up to F31

Messages are backward compatible but older version will only read up to their maximum function number

We currently support up to F31 in trasmission and reception
