# Willkommen zum Traintastic Handbuch

**Traintastic** ist eine Software zur Modellbahnsteuerung.
Mit Traintastic können Lokomotiven, Weichen und Signale sowie anderes Zubehör mit dem Computer gesteuert werden und eine Automatisierung der Modellbahn erfolgen.

Egal ob Du Deine erste digitale Modellbahn baust und eine größere Bahn automatisieren möchtest, Traintastic hilft Dir die Modellbahn einfach und sicher zu steuern.

## Eigenschaften
- Steuerung von Lokomotiven und Zubehör
- Unterstützt mehrere digitale Steuerungssysteme
- Echtzeit Überwachtung Deiner Modellbahn
- Mehrbenutzer und Netzwerk Steuerung
- Erweiterbar mit Lua Scripts

## So arbeitet Traintastic
Traintastic benutzt eine **Client/Server Architektur**:

- Der **Server** steuert Deine Modellbahn und verwaltet die Automatisierung.
- Der **Client** ist die grafische Benutzeroberfläche für den Betrieb und die Überwachung.

Beides kann auf dem selben Computer laufen oder der Server läuft auf einem Computer (zum Beispiel einem Raspberry Pi) und die Clients auf anderen Computern im Netzwerk.
